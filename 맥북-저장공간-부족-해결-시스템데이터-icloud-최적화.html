1️⃣ iCloud Drive에 저장 — 데스크탑·문서 자동 동기화

시스템 설정 → 일반 → 저장 공간 화면에서 「iCloud Drive에 저장」을 활성화하면
데스크탑과 문서 폴더의 파일이 iCloud로 올라가고, 오래 사용하지 않은 파일은 로컬에서 자동 제거됩니다.

⚠️ iCloud 요금제에 충분한 여유 용량이 있을 때만 활성화하세요.
현재 한국 기준 iCloud+ 요금: 50GB ₩1,100/월 · 200GB ₩3,300/월 · 2TB ₩11,000/월

2️⃣ 다운로드 폴더 & 휴지통 비우기

Finder 사이드바에서 다운로드 폴더를 열고 날짜순 정렬 후 오래된 파일부터 삭제합니다.
이후 독(Dock)의 휴지통을 우클릭 → 휴지통 비우기를 반드시 실행합니다.

💡 Finder 상단 메뉴 Finder → 환경설정(Settings) → 고급에서
「30일이 지난 항목을 휴지통에서 자동으로 제거」를 켜두면 이후에는 신경 쓸 필요가 없습니다.

3️⃣ 앱 캐시 수동 삭제 (~/Library/Caches)

Finder를 열고 메뉴바에서 이동 → 폴더로 이동을 클릭하거나 Cmd+Shift+G를 누릅니다.
입력창에 ~/Library/Caches를 입력하고 엔터를 누르면 앱별 캐시 폴더가 나타납니다.

✅ 내부 폴더를 열어 크기가 큰 것 위주로 삭제하고 휴지통을 비웁니다.
✅ 삭제해도 안전합니다 — 앱이 다음 실행 시 자동으로 캐시를 재생성합니다.
✅ 카카오톡 캐시: ~/Library/Caches/com.kakao.KakaoTalkMac 폴더 확인

4️⃣ 대용량 파일 & 사용하지 않는 앱 정리

시스템 설정 → 일반 → 저장 공간에서 「파일 검토」를 누르면
대용량 파일, 다운로드, 지원되지 않는 앱 등을 한눈에 볼 수 있습니다.

☁️ iCloud 사진 최적화 — 가장 빠르게 수십 GB 확보하는 법

맥북 저장 공간의 상당 부분을 차지하는 주범 중 하나가 사진 라이브러리입니다.
iPhone으로 찍은 사진과 영상이 iCloud를 통해 맥북에도 고화질 원본으로 동기화되어 있다면
수십 GB, 많게는 수백 GB까지 잡아먹고 있을 수 있습니다.

iCloud 사진 최적화 설정 방법

① 사진(Photos) 앱 실행
② 메뉴바 Photos → 설정(Settings) 클릭
③ iCloud 탭 → 「Mac 저장 공간 최적화」 선택

이 설정을 켜면 원본 파일은 iCloud에만 보관되고, 맥북에는 저화질 썸네일만 저장됩니다.
사진을 열 때 인터넷에 연결되어 있으면 원본을 바로 내려받아 감상할 수 있습니다.

⚠️ 주의: iCloud 저장 공간이 부족하면 이 기능이 제대로 작동하지 않습니다.
현재 iCloud 사용량은 시스템 설정 → Apple 계정 → iCloud에서 확인하세요.

🚀 정체불명 「시스템 데이터」 안전하게 줄이는 법

시스템 설정의 저장 공간 화면에서 「시스템 데이터」가 50GB, 100GB를 넘어가는 경우가 흔합니다.
이 항목은 macOS가 자동으로 관리하는 파일들인데, 직접 들어가서 정리할 수 있습니다.

Finder에서 Cmd+Shift+G를 눌러 아래 경로들을 직접 확인하세요.

1️⃣ iOS·iPadOS 기기 백업 삭제

아이폰이나 아이패드를 맥북에 케이블로 연결해 백업한 적 있다면
Finder 사이드바에서 해당 기기를 선택 → 상단의 「일반」 탭 → 「백업 관리」를 클릭합니다.
오래된 백업을 선택하고 삭제하면 됩니다.

직접 경로를 찾고 싶다면: ~/Library/Application Support/MobileSync/Backup/

2️⃣ Xcode 시뮬레이터 & 개발 캐시 (개발자용)

Xcode를 사용하는 개발자라면 시뮬레이터 데이터가 수십 GB씩 쌓입니다.
터미널에서 아래 명령어로 사용하지 않는 시뮬레이터를 한 번에 정리하세요.

xcrun simctl delete unavailable

또는 Xcode 앱 → Window → Devices and Simulators → 불필요한 시뮬레이터 삭제

3️⃣ 시스템 로그 & 임시 파일

아래 경로의 파일들은 직접 삭제해도 시스템에 영향이 없습니다.

✅ ~/Library/Caches — 앱 캐시 (앱별 폴더 단위로 삭제)
✅ /private/var/folders — macOS 임시 파일 (재시작 시 자동 정리되므로 건드리지 않아도 됨)
✅ ~/Library/Logs — 앱 로그 파일 (삭제 안전, 문제 진단 불필요하다면 정리 가능)

💡 한 가지 팁: Finder에서 Cmd+Shift+. (마침표)를 누르면
숨김 파일과 폴더가 보여서 Library 내 숨겨진 대용량 폴더를 쉽게 찾을 수 있습니다.

4️⃣ CleanMyMac 없이도 충분한가요?

유료 앱인 CleanMyMac이 편리한 건 사실이지만, 위의 수동 방법으로도
대부분의 경우 20~60GB를 확보할 수 있습니다.

단, 시스템 전체를 자동 스캔하고 안전 여부를 판단해 삭제해 주는 편의성을 원한다면
CleanMyMac X(Setapp 구독 포함) 사용도 합리적인 선택입니다.

❓ 자주 묻는 질문

Q1. CleanMyMac 같은 앱 없이 무료로 50GB 이상 확보할 수 있나요?

충분히 가능합니다. iCloud 사진 최적화, iOS 기기 오래된 백업 삭제, 앱 캐시 정리, Xcode 시뮬레이터 삭제를 모두 실행하면 평균적으로 30~80GB를 확보할 수 있습니다. 특히 오랫동안 정리하지 않은 맥북이라면 100GB 이상 확보되는 경우도 있습니다.

Q2. ~/Library/Caches 폴더 전체를 삭제해도 안전한가요?

기본적으로 안전합니다. 캐시 파일은 앱이 빠르게 실행되도록 임시 저장하는 파일로, 삭제해도 앱이 다음 실행 시 자동으로 다시 생성합니다. 단, 폴더 전체를 한꺼번에 삭제하기보다는 앱별 폴더 단위로 선택적으로 삭제하는 것이 더 안전합니다. 현재 실행 중인 앱의 캐시는 삭제하지 마세요.

Q3. iCloud 사진 최적화를 켜면 기존 사진이 삭제되나요?

사진이 삭제되는 것이 아니라 iCloud에 안전하게 보관됩니다. 맥북에는 저화질 썸네일만 남고, 원본은 iCloud 서버에 있습니다. 인터넷 연결 상태에서 사진을 열면 원본이 다운로드됩니다. 따라서 iCloud 저장 공간에 여유가 충분한 상태에서 켜야 합니다.

Q4. 맥북 재시작 후 저장 공간이 더 늘어날 수 있나요?

네, 맞습니다. macOS는 재시작 시 /private/var/folders의 임시 파일과 swap 파일 일부를 자동 정리합니다. 오랫동안 재시작하지 않은 맥북이라면 재시작만으로 수 GB가 회복되는 경우도 있습니다. 저장 공간 정리 후 재시작을 한 번 해주세요.

Q5. 256GB 맥북, iCloud 없이 버틸 수 있나요?

영상 편집이나 대용량 파일 작업이 없다면, 외장 SSD + 꼼꼼한 로컬 정리로 충분히 가능합니다. 단, 사진 라이브러리가 큰 경우에는 iCloud 사진 최적화 없이는 한계가 있습니다. 사진 수가 많다면 최소 iCloud 200GB(₩3,300/월) 구독을 권장합니다. 3년 비용이 ₩118,800으로 외장 SSD보다 비싸지만 기기 간 동기화와 편의성을 감안하면 충분히 합리적입니다.
