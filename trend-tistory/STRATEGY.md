# 트렌드팩토리 (trend.dokyungja.us) 운영 전략

> **플랫폼**: 티스토리 (애드센스 + 쿠팡파트너스)
> **니치**: 트렌드 (엔터·스포츠·생활정보·디지털팁)
> **현 상태**: 신규
> **목표 6개월차**: 일 방문 1,500+, 월 60~100만 원

---

## 1. 트렌드 니치 = 양날의 검

| 장점 | 단점 |
|------|------|
| 트래픽 폭발 가능 (이슈 발생 시 일 1만 PV) | 빨리 식음 (3~7일) |
| 키워드 진입 쉬움 (속보형) | 도메인 권위 누적 어려움 |
| 쿠팡 전환 다양 (이슈 따라) | 저품질 위험 (펌핑 감지) |

**리스크 관리 핵심**: **트렌드 70% + 에버그린 30%** 비율 유지.
- 트렌드만 쓰면 1년 후 자산 0
- 에버그린 글이 도메인 권위 누적 → 트렌드 글도 빨리 색인

## 2. 발행 카테고리

| 카테고리 | 월 발행 | 비중 | 수익 모델 |
|---------|--------|------|----------|
| **A. 트렌드 이슈** (드라마/스포츠 결과/연예) | 6편 | 40% | 애드센스 (트래픽) |
| **B. 생활 정보** (정부 지원금/제도 변경) | 4편 | 30% | 애드센스 + 정부24 링크 |
| **C. 디지털·앱 팁** (신규 앱 사용법) | 3편 | 20% | 애드센스 + 쿠팡 |
| **D. 시즌 상품 추천** (계절·이벤트) | 2편 | 10% | **쿠팡파트너스 핵심** |

**총 월 15편 (격일 1편)**. 트렌드는 양도 어느 정도 필요.

---

## 3. 트렌드 글 vs 에버그린 글 작성법

### 트렌드 (속보형, 60분/편)
- 본문 1,500~2,000자
- 발생 24~48시간 내 발행 (속도 핵심)
- **팩트체크 필수** (스킬 Phase 0 준수)
- 출처: 방송사·협회·정부 공식
- "확인되지 않음" 표시 적극 활용

### 에버그린 (3시간/편)
- 본문 2,500~3,500자
- 본인 경험·캡처 포함
- 정기 업데이트 (분기마다 갱신)
- 단가 높은 제휴 카테고리 (가전·계절상품)

---

## 4. 키워드 발굴 (트렌드 특화)

### 트렌드 키워드 도구
1. **네이버 데이터랩** — 실시간 검색 트렌드 (시즌·이슈)
2. **구글 트렌드** — 한국 + 글로벌 동향
3. **카카오 인사이트** — 다음 검색 트렌드
4. **trends24.in/south-korea** — 트위터 X 한국 트렌드
5. **유튜브 인기 동영상** — 영상 이슈 → 텍스트화 가능

### 빨리 식는 키워드 회피
- **단발성 사건** (사고·사건): 3일 후 검색 0
- **특정 회차 결과** (예: "결혼하자 8회 결말"): 다음 회 나오면 0
- → 대신 **"~ 정리", "~ 인물 관계도"** 처럼 시리즈 전체 커버

### AI 트렌드 발굴 프롬프트
```
오늘 한국에서 검색 폭증 중인 키워드를 카테고리별 5개씩 제시:
- 엔터: 드라마/예능/연예인 이슈
- 스포츠: 경기 결과/선수 이슈
- 생활정보: 정부 정책/제도 변경
- 디지털: 신규 앱/서비스
각 키워드에 (1) 예상 트래픽 지속 기간 (2) 쿠팡 추천 가능 상품 (3) 
팩트체크 출처 1개
```
**주의**: AI가 모르는 최신 이슈는 못 잡음 → 네이버 데이터랩 / X 트렌드 보조 필수.

## 5. 쿠팡파트너스 결합 (핵심 수익)

트렌드 글 + 쿠팡 = 가장 강력한 조합.

### 자연스럽게 끼워 넣는 패턴
| 트렌드 글 유형 | 자연스러운 쿠팡 연계 |
|--------------|------------------|
| 드라마 화제 → 주인공 패션 | 비슷한 옷·액세서리 |
| 스포츠 결과 → 경기 장비 | 야구글러브, 축구공 |
| 정부 지원금 → 신청 도움 | 공인인증 USB |
| 환절기 알레르기 트렌드 | 공기청정기, 알레르기 약 |
| 신규 앱 사용법 | 관련 액세서리 |

### 최상단 명시 양식 (공정위 표시광고법)
> 💡 **이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다.**

(스킬 파일에 정의된 양식 그대로 사용)

### 글당 쿠팡 링크 배치
- **2~3개** (5개 초과 시 구글 페널티)
- 본문 중간 카드형 1개 + 글 끝 비교 표 1개
- Phase 4 카드형 HTML (스킬 표준)

## 6. 광고 배치
- 카카오 애드핏 + 구글 애드센스 동시
- 트렌드 글: 상단 광고 비중 ↑ (이탈률 높으니 먼저 노출)
- 에버그린 글: 본문 중간 광고 비중 ↑ (체류 길어 클릭 ↑)

## 7. 안티패턴 (트렌드 블로그 사망 패턴)

1. **이슈 펌핑 100%** (이사양잡스 경고): 단기 트래픽 → 영구 저품질
   → **에버그린 30% 의무**로 방어
2. **확인 안 된 루머 발행**: 명예훼손 + 검색 누락
3. **다른 블로그 복붙**: 즉시 색인 누락
4. **연예인 사생활 자극 글**: 카카오/구글 광고 정책 위반
5. **쿠팡 링크 5개 초과**: 페널티
6. **자동 발행 봇 사용**: 영구 차단

## 8. 팩트체크 우선 원칙 (스킬 Phase 0 준수)

트렌드 블로그는 **속도보다 정확성이 장기 자산**:
- ✅ 공식 소스 확인된 정보만 단정형
- ⚠️ 미확인 정보는 "확인되지 않음" 명시
- ❌ 충돌 정보는 제외

→ 6개월 누적 시 "신뢰할 수 있는 트렌드 블로그" 평판 → 검색 우대.
